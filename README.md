# use-fauna

> React hooks for FaunaDB

## Install

```bash
npm install --save use-fauna
```

## Current hooks

- `useDatabase`
- `useUpdateDocument`
- `useDeleteDocument`
- `useGetAll`
- `useGet`
- `useCreate`

---

This hook is created using [create-react-hook](https://github.com/hermanya/create-react-hook).
